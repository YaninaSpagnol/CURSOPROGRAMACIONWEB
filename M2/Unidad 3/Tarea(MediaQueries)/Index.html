Media Queries

Media Queries significa "consulta de medios" y son utiles cuando deseamos modificar el aspecto de nuestra pagina basados en la resolucion de la pantalla o el ancho del viewport del navegador.

Forma alternativa de definir nuestro Media Queries

Una forma no tan usada, pero muchas veces mas beneficiosa para el usurio, de definir nuestos media queries es usando el atributo media de la etiqueta link que usamos para incluis nuestros estilos.
